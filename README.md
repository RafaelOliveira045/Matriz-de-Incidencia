# Matriz-de-Incidencia
Sistema interativo em C para representação de grafos não direcionados via Matriz de Incidência (vértices x arestas). O software permite inicializar o grafo, inserir arestas, exibir a matriz formatada, calcular e listar o grau de cada vértice e mapear as conexões ativas. Homologado com cenário de 6 vértices e 7 arestas para validação lógica.
